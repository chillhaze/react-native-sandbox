Стили
---------------
В RN есть встроенный компонент StyleSheet который нужно импортировать из RN:

import { StyleSheet } from "react-native";

У него есть встроенное свойство 'create', в это свойство
передаем объект с стилями, записанными в JS формате (camelCase).

Всем встроенным компонентам РН уже задан по умолчанию display: flex.
В свойство 'flex' передаем сколько места на экране будет занимать компонент. Если 1 - то весь экран, если компонентов 3, у первого флекс=1, у второго 2, у третьего 3, это означает, в сумме 6. Следовательно компоненты займут такие части экрана: первый 1/6, второй 2/6, третий 3/6.

По умолчанию ориентация задана флексом, как flexDirecton: column.
Значит по вертикали передвигаем компоненты с помощью justifyContent,
а по горизонтали (лево/право) - alignItems.
